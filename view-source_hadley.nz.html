<h2>Teaching</h2>
<hr />
</div>
<div class="col-lg-12">
<p>If you’d like to learn more about what I do, and how to use R effectively, I’d recommend starting with one of my books:</p>
<ul>
<li><p><a href="http://r4ds.had.co.nz">R for Data Science</a>, with Garrett Grolemund, introduces the key tools for doing data science with R.</p></li>
<li><p><a href="http://amzn.com/0387981403?tag=ggplot2-20">ggplot2: elegant graphics for data analysis</a> shows you how to use ggplot2 to create graphics that help you understand your data.</p></li>
<li><p><a href="http://adv-r.hadley.nz">Advanced R</a> helps you master R as a programming language, teaching you what makes R tick.</p></li>
<li><p><a href="http://r-pkgs.org">R packages</a> teaches good software engineering practices for R, using packages for bundling, documenting, and testing your code.</p></li>
</ul>

<p>I also teach in person workshops from time-to-time; see the <a href="https://www.rstudio.com/workshops/">RStudio workshops page</a> for more details.</p>

<section id="code">
<div class="container">
<div class="row">
<div class="col-lg-12 text-center">
<h2>Code</h2>
<hr />
<p>Most of my work is in the form of open source R code, which you can find on <a href="http://github.com/hadley/">my github</a>. You can roughly divide my work into three categories: tools for data science, tools for data import, and software engineering tools.</p>

</div>
</div>
<div class="row">
<div class="col-md-4">
<h3>Data science with the <a href="https://tidyverse.org">tidyverse</a></h3>
<ul>
<li><a href="https://ggplot2.tidyverse.org">ggplot2</a> for visualising data.</li>
<li><a href="https://dplyr.tidyverse.org">dplyr</a> for manipulating data.</li>
<li><a href="https://tidyr.tidyverse.org">tidyr</a> for tidying data.</li>
<li><a href="https://stringr.tidyverse.org">stringr</a> for working with strings.</li>
<li><a href="https://lubridate.tidyverse.org">lubridate</a> for working with date/times.</li>
</ul>
</div>
<div class="col-md-4">
<h3>Data import</h3>
<ul>
<li><a href="https://readr.tidyverse.org">readr</a> for reading .csv and fwf files.</li>
<li><a href="https://readxl.tidyverse.org">readxl</a> for reading .xls and .xlsx files.</li>
<li><a href="https://haven.tidyverse.org">haven</a> for SAS, SPSS, and Stata files.</li>
<li><a href="https://httr.r-lib.org">httr</a> for talking to web APIs.</li>
<li><a href="https://rvest.tidyverse.org">rvest</a> for scraping websites.</li>
<li><a href="https://xml2.r-lib.org">xml2</a> for importing XML files.</li>
</ul>
</div>
<div class="col-md-4">
<h3>Software engineering</h3>
<ul>
<li><a href="http://devtools.r-lib.org/">devtools</a> for general package development.</li>
<li><a href="http://github.com/klutometis/roxygen">roxygen2</a> for in-line documentation.</li>
<li><a href="http://testthat.r-lib.org/">testthat</a> for unit testing</li>
<li><a href="https://pkgdown.r-lib.org">pkgdown</a> to create beautiful package websites</li>
</ul>
</div>
</div>
</div>
</section>
